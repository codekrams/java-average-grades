# java-average-grades

Schreiben sie ein Programm, dass einen Nutzer auffordert Noten zwischen 1 und 6 einzugeben (mit Eingabeüberprüfung auf den korrekten Wert  und eventueller Fehlermeldung bei Falscheingabe), die Eingaben soll mit 0 abgeschlossen werden. 

Abschließend soll der Notenschnitt berechnet werden.
